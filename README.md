# telephony.museum

Static website for the Telephony Museum in Å iauliai, Lithuania. Built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com).

## Development

```sh
npm install       # Install dependencies
npm run dev       # Start dev server at localhost:4321
npm run build     # Build production site to ./dist/
npm run preview   # Preview the build locally
```
